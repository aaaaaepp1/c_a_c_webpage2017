EscapeVR班の活動報告！

はじめに

　こんにちは、escapeVR班です！暑い日が続きますがみなさんいかがお過ごしでしょうか。さて、このC.A.C web siteでは定期的にプロジェクトの紹介をしていくということで、その一番手として僕たちが立ち上げたプロジェクトであるescapeVR班の紹介をしていきたいと思います！

プロジェクトの概要（担当 ：にしお）

　簡単に説明すると、、、このプロジェクトは脱出ゲームが作りたいというにしお君の提案が、VRで作ったら面白くね？というOkamoto君のアイデアによって、ただの脱出ゲームではない、VR脱出ゲームへと変貌を遂げたーというプロジェクトとなっております。
　プロジェクトのメンバーは、にしお、Okamoto、森長、宮崎先輩の4人で、週一で集まって会議をしたりなど、日々活動しています！

EscapeVRについて（担当：Okamoto）

　今僕たちが作っているEscapeVRは、今までブラウザ中心で発展してきた「脱出ゲーム」をより没入感、現実味のあるバーチャルリアリティでプレイできるゲームです！
　ゲーム自体の開発はゲームエンジンであるUnityを使用し、ゲームで使用するモデルはBlenderなど様々なソフトを使っています。今は作らないといけないモデルが半分くらい、ソースコードは2割くらいの完成度です。
　部活でのプロジェクトの目標は、学祭で展示するゲームなどを作ることですが、このプロジェクトはISCAの提出を目標に開発をしています！コンテストの締め切りが8月いっぱいなので、みんな必死になって開発を進めています。

作業について（担当：森長）

　EscapeVR班では週に１回メンバー全員でミーティングをして、それぞれの進捗確認や、今後の方針などを話し合っています。脱出ゲーム大好きなにしおくんと私を中心に、脱出の仕組みもみんなで話し合って考えましたϵ( ‘Θ’ )϶
ゲームに登場するモデル・サウンド・プログラムも全部自分達で作っています！
メンバーの作業の様子や作品を紹介していきます^o^

にしおくんの椅子のモデリングの作業の様子です！
モデリングはblenderを使って行っています。
初めてのモデリングらしいですが、他にもたくさんモデルを作ってくれていますp(^_^)q

他にもサウンド制作やシステム設計、GUIデザインなど、様々な作業をメンバーで分担しています！

EscapeVRのこれから（担当：Okamoto）

　今後は、計画通りだと残りのモデルと、プログラムは先にユーザインターフェースまわりのシステムから開発していく予定です。ISCAで提出するプレゼン用の映像も仕上げないと・・・。頑張ります。でも、8月いっぱいで動くものができるので、夏合宿ではすごい進捗が言えそう。みんなをびびらせてやるぜ(‘-‘*)ﾌﾌ
